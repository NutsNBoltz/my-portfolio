Java Chess Game – Overview

This project is a fully functional two-player chess game written in Java. The sprites for the chess pieces were taken from an open source library. It features valid move detection, and a clean interface. It is my first ever project that I made. It is far from good, but I intend to improve it.

And for the future, I plan to add different gamemodes, as well as Computer Engine. Stay Tuned. There are ofcourse some flaws even in this. For example:

  • Certain conditions of stalemate do not work
  • It does not check for checkmate after Promotion, so sometimes checkmate occurs, but game over message does not display

Note: I'm pretty sure, the file will not run on linux. It is only working on windows as per my observation. Linux version coming soon :D

Tic Tac Toe – Python Project

This is a simple GUI-based Tic Tac Toe game created with Python and Tkinter. Players can choose between a 3×3 or 4×4 grid, and the game allows two players to play locally. Very simple setup. I was planning on making an arcade type GUI where players can choose from different games, just as atari, tetris, pong, but I didnt have time for all that. Possible improvements:

  • Add a custom size for the board
  • Have custom win conditions that players can define
  • Add a bot, for multiplayer purpose.
