Java Chess Game – Overview

This project is a fully functional two-player chess game written in Java. The sprites for the chess pieces were taken from an open source library. It features valid move detection, and a clean interface. It is my first ever project that I made. It is far from good, but I intend to improve it.

And for the future, I plan to add different gamemodes, as well as Computer Engine. Stay Tuned. There are ofcourse some flaws even in this. For example:

  • Certain conditions of stalemate do not work
  • It does not check for checkmate after Promotion, so sometimes checkmate occurs, but game over message does not display
